Tutorial em Programção de Computadores


Programação em C não é difícil mas é composta por várias regras. Um programa em C é composto por um conjunto de Funções. Algumas regras iniciais para se ter em mente:

- A função pela qual o programa começa a ser executado chama-se main.

- Após cada cada comando em C deve-se colocar um ; (ponto-e-vírgula).

Um programa em C deve ser Identado para que possa ser lido com mais facilidade. Indentação é o processo de alinhar todos os comandos nas colunas apropriadas.

Para colocar uma linha de comentário, ou seja, uma linha que o programa não leia e sirva apenas para orientação do programador é só colocar "//" e o texto depois.


Variáveis


Uma Variável é uma posição de memória que pode ser identificada através de um nome.

Podem ter seu conteúdo alterado por um comando de atribuição.

Após a atribuição mudam de valor. Exemplo:

int a, b; // Aonde "int" é o tipo de variável e "a, b" são 2 variáveis

a = 3; // Aqui a variável "a" está recebendo(=) o valor 3

b = a * 2; // Variável "b" recebe o dobro do valor de "a", então "b" recebe o valor 6

c = a + b + 2; // A variável c está recebendo "a" + "b" + 2, então nesse caso seria 3 + 6 + 2 = 11


Tipos de Variáveis


Todas as variáveis em C tem um tipo.

Cada tipo define os valores que a variável pode armazenar.

Cada tipo ocupa uma certa quantidade de memória.

Tipo Valores Válidos
char(%c) Letras e símbolos:'a','b','H','^','*','1','0'
int(%d) Conjunto dos números inteiros
float(%f) Conjunto dos números reais

Entrada e Saída


Para que o usuário do programa consiga entender a finalidade do programa e consiga utiliza-lo é preciso que seja feita a "saída" de informações e a "entrada" de dados.

A entrada de dados (ou leitura) no C é feita através do comando "scanf" e a saída com o comando "printf".

O printf deve ter um parentêses após ele e o texto deve ser digitado entre aspas: printf("Texto teste");

Já o scanf é mais complicado por ser preciso declarar a variável que será "lida" e por ser preciso também usar o indicador(&) antes da variável: Supondo que a variável "exemplo" é do tipo "char" --> scanf("%c",&exemplo);

Agora, quando for utilizar o printf com alguma variável que foi "lida" será preciso declarar o tipo de variável assim como no scanf, mas como é um texto, essa variável deverá ficar no lugar de onde ficaria a palavra. Exemplo:

			
main(){

			
	float valor;

			
	char aluno;

			
	printf("Digite o nome do aluno:");

			
	scanf("%c", &aluno);

			
	printf("Digite a nota de matematica do aluno:");

			
	scanf("%f", &valor);

			
	printf("O %c está com %f em matematica.",aluno,valor);

			
 	//Aonde "%c" será o nome do aluno e "%f" será sua nota.
			
}

Atribuição


Atribuição ou "receber" um valor é como foi demonstrado no exemplo anterior quando você altera o valor de uma variável. Essa atribuição pode ser feita a qualquer momento do programa. Voltando no exemplo anterior, "a" recebeu o valor "3" mas poderia adicionar outra linha de comando em algum momento que atribui outro valor a "a".

int a, b;

a = 3;

b = a * 2;

c = a + b + 2;

a = 20 // Agora "a" que tinha o valor "3", teve seu valor alterado pra "20".


Condição

A condição na linguagem C é definida como uma expressão que poder verdadeira ou falsa.

Exemplo: (3>2) --> Verdadeiro, porque 3 é maior que 2.

Os operadores relacionais usados no C são:

== Igual a
!= Diferente de
>= Maior ou igual
> Maior que
< Menor que
<= Menor ou igual

Existem também os operadores lógicos:

|| OU lógico
&& E lógico
! Negação

SE e SENÃO


O "se" e "senão" é muito importante para fazer condições no C. Na Linguagem C o "se" é representado por "if" e o "senão" é representado por"else". Exemplo de um programa usando "if else":

			
main(){

			
	int num;

			
	printf("Digite um numero, se for menor que 10 a cor e Azul, senão a cor sera Verde:);

			
	scanf("%d",&num);

			
	if(num < 10){

			
		printf("Cor Azul")

			
	}else

			
		printf("Cor Verde")

			
}

Repetição


O comando "while" no C permite que um trecho do programa seja executado ENQUANTO uma certa condição for VERDADEIRA. Sua estrutura funciona da seguinte maneira:

while(condição){

//Os comandos que serão executados ficam aqui.

//Lembrando que podem ser mais que uma linha de comando.

}

//E aqui podem ser colocados outros comandos caso a condição do "while" se torne FALSA.

Ou seja, supondo que o usuário tenha que digitar algum número(x) e se esse número digitado for 1, que apareça a mensagem 123456, a estrutura ficaria dessa forma:

			
while(x == 1){

			
	printf("123456");

			
}
